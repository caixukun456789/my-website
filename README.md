## 当前文件库怎么工作

当前版本使用浏览器 IndexedDB 保存文件。

简单说：

- 文件保存在你当前打开网页的浏览器里。
- 文件不会上传到 GitHub。
- 别人打开公网网站，看不到你浏览器里保存的文件。
- 换电脑、换浏览器，或者清理浏览器数据后，文件可能看不到。

## 安全提醒

不要把身份证、密码、合同、隐私论文、账号资料等敏感文件保存进去。

如果你想让不同电脑都能看到同一批文件，需要后续增加：

- 登录系统
- 云端数据库
- 云端文件储存
- 权限控制

## 主要文件

- `index.html`：网页结构和文字
- `styles.css`：页面样式
- `script.js`：本地文件库功能
- `site-data.json`：网站信息备忘
