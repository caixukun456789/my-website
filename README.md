# 我的个人网站

这是一个本地静态网站，可以后续继续修改。

## 怎么打开

双击这个文件即可预览：

`C:\Users\28168\.codex\第一\my-website\index.html`

如果你想用本地服务器预览，可以在这个文件夹运行：

```powershell
python -m http.server 5173
```

然后打开：

`http://localhost:5173`

## 怎么修改内容

主要改这几个文件：

- `index.html`：网页文字和栏目。
- `styles.css`：颜色、字体、布局、间距。
- `script.js`：菜单、复制路径等小交互。
- `site-data.json`：记录以后要替换的内容，不影响当前页面显示。
